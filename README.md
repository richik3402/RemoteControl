# RemoteControl
RemoteControl — это современная программа для удалённого управления компьютерами по локальной сети, написанная на C# и Avalonia UI.

Приложение позволяет:
📺 транслировать экран удалённого компьютера,
🖱 управлять мышью,
⌨ управлять клавиатурой с поддержкой Unicode и русской раскладки,
📸 делать снимки экрана,
📋 работать с буфером обмена,
⚡ удалённо выключать и перезагружать ПК,
⚙ гибко настраивать качество и FPS трансляции.

Возможности:
поддержка Windows,
клиент-серверная архитектура,
передача изображений через TCP,
настройка качества стрима,
управление несколькими клиентами,
сохранение устройств,
настройки прав доступа для мыши и клавиатуры,
современный интерфейс на Avalonia UI

Технологии:
C#,
.NET,
Avalonia UI,
TCP Sockets,
JSON,
WinAPI (SendInput).

RemoteControl is a modern remote desktop application for controlling computers over a local network, built with C# and Avalonia UI.

The application allows you to:
📺 stream the remote screen,
🖱 control the mouse,
⌨ control the keyboard with full Unicode and multilingual support,
📸 capture screenshots,
📋 synchronize the clipboard,
⚡ remotely shut down or restart PCs,
⚙ configure stream quality and FPS.

Features:
Windows support,
client-server architecture,
TCP-based image streaming,
adjustable stream quality,
multi-client management,
saved devices list,
mouse and keyboard permission settings,
modern Avalonia UI interface.

Technologies
C#,
.NET,
Avalonia UI,
TCP Sockets,
JSON,
WinAPI (SendInput).
